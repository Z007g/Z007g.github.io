变量上浮：在一个方法或者全局代码中，我们在生命变量之前使用变量时并没有抛异常，而是返回undefined。这是因为javascript自动把变量的声明上浮到函数或者全局的最前面。如下面的代码:

/**
* 全局变量上浮
*/
console.log(x === undefined); // logs "true"
var x = 3;

/**
* 方法变量上浮
*/
var myvar = "my value";
// 打印变量myvar结果为：undefined
(function() {
console.log(myvar); // undefined
var myvar = "local value";
})();
